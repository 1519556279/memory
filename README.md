# Memory Skill — 全局记忆系统

一个通用的 AI 长期记忆技能。自动记录你的偏好和关键事实、压缩旧对话节约 Tokens、跨会话恢复记忆。

让你和 AI 的对话不再遗忘，同时大幅减少 Token 消耗。

---

## 解决的问题

**AI 没有持久记忆。** 每次新对话都是全新的，导致：

- 你要反复说同样的需求和偏好
- 写到一半关掉，下次就忘了
- 对话长了，最早的内容被丢弃

**记忆技能通过「文件即记忆」解决：所有记忆存硬盘，新对话自动恢复。**

---

## 功能特性

### 通用记录，不分领域
不再区分代码/小说/日常对话。你说过的**所有重要信息**都会被记录：
- **偏好**：你喜欢 TypeScript / 2 空格缩进 / 仙侠小说 / 简洁设计
- **事实**：你删除了某个功能 / 做了某个决定 / 项目进度
- **上下文**：你在做什么、你的目标是什么

### 自动压缩上下文，节约 Tokens
对话超过一定轮次后，最早的对话自动压缩为摘要，保留关键信息扔掉冗余内容。

`
传统方式：从头聊 → 我啥都不记得 → 你重复说 → 浪费 Tokens
记忆方式：恢复记忆 → 直接继续 → 省 Tokens
`

### 跨对话恢复
今天聊完关掉，明天新对话自动加载记忆，你不需要重新说一遍。

### 关键词索引检索
每条记忆自动标注标签和关键词。说「记得我上次说过…」就能搜到。

### 按项目隔离
A 项目的记忆不会混到 B 项目。

## 安装

### 方式一：直接复制

将 \skills/memory/\ 放到 AI 的 skills 目录即可。

### 方式二：reasonix 环境

`powershell
install_skill(scope:"global")
`

## 文件结构

\\\
skills/memory/
├── SKILL.md          # 主技能文件（核心指令）
├── REFERENCE.md      # 使用参考
├── README.md         # 本文件
└── LICENSE           # MIT
\\\

### 记忆数据存储位置

\\\
~/.reasonix/memory/
└── projects/
    └── <项目标识>/
        ├── preferences.md      ← 你的偏好/习惯/风格（自动维护）
        ├── facts.md            ← 关键事实/决策/状态（自动维护）
        ├── index.json          ← 标签+关键词索引（自动维护）
        └── archive.jsonl       ← 压缩后的旧对话摘要
\\\

每个项目独立存储，互不干扰。

## 使用方式

### 自动工作（无需操作）

- 你说「我喜欢用 TypeScript」→ 自动记到 preferences.md
- 你说「删掉 xxx 功能」→ 自动记到 facts.md
- 新对话开始 → 自动加载记忆 → 输出恢复报告

### 手动命令

| 命令 | 作用 | 示例 |
|------|------|------|
| \/记住 内容\ | 手动保存一条偏好或事实 | \/记住 我喜欢用 pnpm\ |
| \/忘记 关键词\ | 删除包含关键词的记忆 | \/忘记 pnpm\ |
| \/记忆\ | 查看所有记忆摘要 | \/记忆\ |
| \/压缩记忆\ | 手动触发对话压缩 | \/压缩记忆\ |
| \/清理记忆\ | 整理低价值记忆 | \/清理记忆\ |

## 工作流程

\\\
你开口说话
    ↓
自动记录偏好/事实 → preferences.md + facts.md
    ↓
建立关键词索引 → index.json
    ↓
            ← 对话超过阈值？
            ↓ 是
    压缩旧对话 → archive.jsonl（释放上下文）
    ↓
新对话开始
    ↓
读取记忆文件
    ↓
输出恢复报告：「已恢复 N 条偏好，M 条事实」
    ↓
直接继续，无需重复
\\\

### 压缩策略

| 对话轮次 | 行为 |
|---------|------|
| 0-50 轮 | 保持完整对话 |
| 50-100 轮 | 最早的 20 轮压缩为摘要，保留最近 30 轮原文 |
| 100+ 轮 | 继续压缩，永远保留最近 30 轮原文 |
