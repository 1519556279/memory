# Memory Skill - 全局记忆系统

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个通用的 AI 长期记忆技能。自动记录偏好和事实、压缩旧对话节约 Tokens、跨会话恢复记忆。

让你和 AI 的对话不再遗忘，同时大幅减少 Token 消耗。

---

## 解决的问题

AI 没有持久记忆。每次新对话都是全新的，导致：

- 你要反复说同样的需求和偏好
- 写到一半关掉，下次就忘了
- 对话长了，最早的内容被丢弃

**记忆技能通过「文件即记忆」解决：所有记忆存硬盘，新对话自动恢复。**

## 功能特性

### 通用记录，不分领域

不再区分代码/小说/日常对话。所有重要信息都会被记录：

- **偏好**：你喜欢 TypeScript / 2 空格缩进 / 仙侠小说 / 简洁设计
- **事实**：你删除了某个功能 / 做了某个决定 / 项目进度
- **上下文**：你在做什么、你的目标是什么

### 自动压缩上下文，节约 Tokens

对话超过一定轮次后，最早的对话自动压缩为摘要，保留关键信息扔掉冗余内容。

### 跨对话恢复

今天聊完关掉，明天新对话自动加载记忆，你不需要重新说一遍。

### 关键词索引检索

每条记忆自动标注标签和关键词。说「记得我上次说过...」就能搜到。

### 按项目隔离

A 项目的记忆不会混到 B 项目。

## 使用方式

### 自动工作（无需操作）

- 你说「我喜欢用 TypeScript」-> 自动记到 preferences.md
- 你说「删掉 xxx 功能」-> 自动记到 facts.md
- 新对话开始 -> 自动加载记忆 -> 输出恢复报告

### 手动命令

| 命令 | 作用 | 示例 |
|------|------|------|
| /记住 内容 | 手动保存偏好或事实 | /记住 我喜欢用 pnpm |
| /忘记 关键词 | 删除记忆 | /忘记 pnpm |
| /记忆 | 查看所有记忆 | /记忆 |
| /记忆 #标签 | 按标签过滤 | /记忆 #代码 |
| /压缩记忆 | 手动压缩对话 | /压缩记忆 |
| /清理记忆 | 整理低价值记忆 | /清理记忆 |
