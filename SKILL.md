# Memory — 全局记忆系统

## 本技能是什么

一个通用的 AI 长期记忆系统。解决 AI 没有持久记忆的问题。

| 解决的问题 | 效果 |
|-----------|------|
| 你反复说同样的需求 | 一次记录，永久记住 |
| 写到一半关掉就忘了 | 跨会话自动恢复 |
| 对话太长最早的内容被丢弃 | 自动压缩旧对话，节约 Tokens |
| 不记得之前做过什么决定 | 关键词索引，一搜就到 |

## 快速开始

### 新对话启动（自动）

检测 `~/.reasonix/memory/projects/<project-id>/` 是否存在记忆文件。
如果存在，加载并输出恢复报告：

```
🧠 记忆已恢复
━━━━━━━━━━━━━━━━━━
📋 偏好记录: N 条
📌 关键事实: N 条
📦 压缩存档: N 条
━━━━━━━━━━━━━━━━━━
```

### 自动保存（无需操作）

| 你说的话 | 自动保存到 |
|---------|-----------|
| "我喜欢用 TypeScript" | preferences.md |
| "把这个功能删掉" | facts.md |
| "已经写到第 5 章了" | facts.md |
| "我用 pnpm 不用 npm" | preferences.md |

### 手动命令

| 命令 | 作用 | 示例 |
|------|------|------|
| `/记住 内容` | 手动保存偏好或事实 | `/记住 我喜欢用 pnpm` |
| `/忘记 关键词` | 删除记忆 | `/忘记 pnpm` |
| `/记忆` | 查看所有记忆 | `/记忆` |
| `/记忆 #标签` | 按标签过滤 | `/记忆 #代码` |
| `/压缩记忆` | 手动触发对话压缩 | `/压缩记忆` |
| `/清理记忆` | 整理低价值记忆 | `/清理记忆` |

## 模块索引

本技能由以下模块组成，按需加载：

| # | 模块 | 用途 | 触发条件 |
|---|------|------|---------|
| 01 | modules/01-core.md | 核心概念与初始化 | 新对话/加载时 |
| 02 | modules/02-storage.md | 记忆文件格式 | 保存/读取记忆时 |
| 03 | modules/03-index.md | 索引与搜索 | 检索记忆时 |
| 04 | modules/04-compression.md | 上下文压缩 | 对话超过阈值/手动触发 |
| 05 | modules/05-retrieval.md | 检索与注入 | 用户问"记得..."时 |
| 06 | modules/06-commands.md | 手动命令 | 用户输入命令时 |

## 记忆文件结构

```
~/.reasonix/memory/
└── projects/
    └── <project-id>/
        ├── preferences.md     <- 偏好（自动维护）
        ├── facts.md           <- 事实（自动维护）
        ├── index.json         <- 索引（自动维护）
        └── archive.jsonl      <- 存档（压缩时写入）
```

## 配合其他 Skills

| Skill | 配合方式 |
|-------|---------|
| tianming-skill | 记住小说进度/角色设定/写作风格 |
| frontend-design-color | 记住配色偏好/设计风格 |
| 任何 skill | 记住用户的所有偏好和习惯 |

## 完整工作流程

```
新对话开始
    ↓
加载 01-core.md → 检测记忆文件
    ↓
有记忆？ → 加载 02-storage.md → 读取文件 → 输出恢复报告
    ↓
用户说话
    ↓
匹配到"记得/之前/说过"？ → 加载 03-index.md + 05-retrieval.md → 搜索 → 注入上下文
    ↓
匹配到偏好/事实？ → 加载 02-storage.md → 保存到文件 → 更新索引
    ↓
超过 50 轮？ → 加载 04-compression.md → 压缩旧对话 → 写入 archive.jsonl
    ↓
用户输入命令？ → 加载 06-commands.md → 执行命令
```

## 关于 Token 节约

| 对话轮次 | 原始估算 | 压缩后 | 节省 |
|---------|---------|--------|------|
| 50 轮 | ~15,000 | ~12,000 | ~20% |
| 100 轮 | ~30,000 | ~18,000 | ~40% |
| 200 轮 | ~60,000 | ~25,000 | ~58% |

## 存储位置

全局: `~/.reasonix/memory/projects/<project-id>/`
项目: 按项目 ID 隔离，互不干扰
