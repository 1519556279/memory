# Module 03: Index - 索引与检索

## 索引结构

每条记忆在 index.json 中记录：

```json
{
  "id": "唯一标识",
  "content": "摘要内容",
  "tags": ["标签1", "标签2"],
  "date": "2024-06-07",
  "keywords": ["关键词1", "关键词2"]
}
```

## 标签系统

| 标签 | 含义 | 示例 |
|------|------|------|
| #代码 | 编程相关 | TypeScript, React |
| #小说 | 写作相关 | 角色, 世界观 |
| #偏好 | 用户喜好 | 风格, 习惯 |
| #决定 | 关键决策 | 安装, 删除 |
| #项目 | 项目状态 | 进度, 里程碑 |

## 检索方式

### 关键词搜索

当你说「记得之前说过...」时：

```
1. 提取搜索词
2. 在 index.json 的 keywords 字段中匹配
3. 在 preferences.md 和 facts.md 中全文搜索
4. 按日期倒序返回结果
```

### 标签过滤

```
/记忆 #代码      -> 只看代码相关的记忆
/记忆 #小说      -> 只看小说相关的记忆
/记忆 #偏好      -> 只看偏好记录
```

### 搜索结果排序

优先级从高到低：

1. 标签精确匹配
2. 关键词匹配（index.json）
3. 全文匹配（preferences.md / facts.md）
4. 日期倒序（最新的排前面）
