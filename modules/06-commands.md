# Module 06: Commands — 手动命令

## 命令速查表

| 命令 | 作用 | 快速示例 |
|------|------|---------|
| `/记住 内容` | 手动保存偏好或事实 | `/记住 我喜欢用 pnpm` |
| `/忘记 关键词` | 删除包含关键词的记忆 | `/忘记 pnpm` |
| `/记忆` | 查看所有记忆摘要 | `/记忆` |
| `/记忆 #标签` | 按标签查看 | `/记忆 #代码` |
| `/压缩记忆` | 手动触发对话压缩 | `/压缩记忆` |
| `/清理记忆` | 整理低价值记忆 | `/清理记忆` |

---

## 命令一：/记住

### 用途

手动保存一条偏好或事实到记忆文件。

### 格式

```
/记住 [内容]
```

### 自动分类规则

| 内容特征 | 保存到 | 标签 |
|---------|--------|------|
| 包含"喜欢/不喜欢/用/习惯/风格" | preferences.md | 偏好 |
| 包含"装了/删了/决定/进度" | facts.md | 事实 |
| 不确定 | preferences.md | 通用 |

### 示例

```
用户：/记住 我喜欢用 pnpm 而不是 npm
→ 分类：偏好
→ 写入 preferences.md
→ 更新 index.json
→ 回复：✅ 已记住：你喜欢用 pnpm

用户：/记住 我今天删除了 computer-control
→ 分类：事实
→ 写入 facts.md
→ 更新 index.json
→ 回复：✅ 已记住：删除了 computer-control
```

## 命令二：/忘记

### 用途

删除包含指定关键词的记忆。

### 格式

```
/忘记 [关键词]
```

### 执行流程

```
1. 在 index.json 的 keywords 中搜索匹配项
2. 在 preferences.md 中删除相关行
3. 在 facts.md 中删除相关行
4. 从 index.json 中删除相关条目
5. 重建索引
```

### 示例

```
用户：/忘记 pnpm
→ 搜索 index.json -> 找到 pref-003
→ 从 preferences.md 删除相关行
→ 从 index.json 删除条目
→ 回复：✅ 已忘记：pnpm（1 条）

用户：/忘记 第3章
→ 搜索所有文件 -> 找到 2 条匹配
→ 删除匹配行
→ 重建索引
→ 回复：✅ 已忘记：第3章（2 条）
```

## 命令三：/记忆

### 用途

查看当前所有记忆的摘要。支持标签过滤。

### 格式

```
/记忆           ← 查看全部
/记忆 #代码     ← 只看代码相关
/记忆 #小说     ← 只看小说相关
/记忆 #偏好     ← 只看偏好记录
```

### 输出格式

```
用户：/记忆
──────────────────────────────────
🧠 当前记忆

📋 偏好: 3 条
  1️⃣ TypeScript，2 空格
  2️⃣ 仙侠小说，主角林峰
  3️⃣ Apple 简洁设计风格

📌 事实: 4 条
  1️⃣ 删除了 novel-memory-manager
  2️⃣ 安装了 tianming-skill
  3️⃣ 删除了 computer-control
  4️⃣ 安装了 memory skill

📦 存档: 2 条摘要（最近: 2024-06-07）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
用户：/记忆 #代码
──────────────────────────────────
📋 代码相关记忆
  1️⃣ TypeScript，2 空格缩进
  2️⃣ 用 pnpm 代替 npm

```

## 命令四：/压缩记忆

### 用途

手动触发对话压缩。

### 执行流程

```
1. 检查当前对话已进行的轮次
2. 如果有超过 30 轮前的旧内容
3. 对最早的 XX 轮生成摘要
4. 写入 archive.jsonl
5. 输出压缩报告
```

### 输出格式

```
📦 压缩完成
· 压缩范围：第 1-20 轮
· 保存 Token：~3,200
· 存档位置：archive.jsonl
```

---

## 命令五：/清理记忆

### 用途

整理低价值记忆，释放索引空间。

### 执行流程

```
1. 扫描 index.json
2. 标记 importance < 3 的条目
3. 标记超过保留期限的条目
4. 从 preferences.md 和 facts.md 中删除对应行
5. 重建 index.json
6. 清理 archive.jsonl 中重要性低的条目
```

### 输出格式

```
🧹 清理完成
· 删除低价值条目：5 条
· 保留有效记忆：12 条
· 重建索引：完成
```
