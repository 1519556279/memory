# Module 06: Commands - 手动命令

## 命令列表

| 命令 | 作用 | 示例 |
|------|------|------|
| /记住 | 手动保存偏好或事实 | /记住 我喜欢用 pnpm |
| /忘记 | 删除记忆 | /忘记 pnpm |
| /记忆 | 查看所有记忆 | /记忆 |
| /记忆 #代码 | 按标签查看 | /记忆 #代码 |
| /压缩记忆 | 手动压缩对话 | /压缩记忆 |
| /清理记忆 | 整理低价值记忆 | /清理记忆 |

## /记住 命令

手动保存一条偏好或事实。

```
用户：/记住 我喜欢用 pnpm 而不是 npm
结果：写入 preferences.md -> 更新 index.json

用户：/记住 我今天删除了 computer-control skill
结果：写入 facts.md -> 更新 index.json
```

## /忘记 命令

删除包含关键词的记忆。

```
用户：/忘记 pnpm
结果：搜索 index.json -> 从 preferences.md 删除 -> 更新索引

用户：/忘记 第3章
结果：搜索 facts.md -> 删除匹配行 -> 重建索引
```

## /记忆 命令

查看所有记忆摘要，支持标签过滤。

```
用户：/记忆
输出：
🧠 当前记忆
━━━━━━━━━━━━━━━━━━
📋 偏好: 3 条
   ① TypeScript，2 空格
   ② 仙侠小说，主角林峰
   ③ Apple 简洁设计风格

📌 事实: 4 条
   ① 删除了 novel-memory-manager
   ② 安装了 tianming-skill
   ③ 删除了 computer-control
   ④ 安装了 memory skill

📦 存档: 2 条摘要
━━━━━━━━━━━━━━━━━━
```

## /压缩记忆 命令

手动触发对话压缩。

```
用户：/压缩记忆
结果：检查当前对话轮次 -> 超过 50 轮则压缩 -> 写入 archive.jsonl
```

## /清理记忆 命令

整理低价值记忆。

```
用户：/清理记忆
结果：扫描所有记忆 -> 删除重要性低且超过 30 天的条目 -> 重建索引
```
