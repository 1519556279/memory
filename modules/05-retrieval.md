# Module 05: Retrieval - 记忆检索与注入

## 什么时候检索

| 场景 | 触发词 | 检索范围 |
|------|--------|---------|
| "我之前是不是说过..." | 记得/之前/说过 | index.json + 全文 |
| "我的偏好是什么" | 偏好/喜欢/习惯 | preferences.md |
| "我们做了什么决定" | 决定/之前/删了/装了 | facts.md |
| 新对话开始 | 自动 | 全部记忆文件 |
| 用户手动 | /记忆 | 全部 |

## 检索流程

```
收到用户消息
    ↓
匹配触发词？
    ↓ 是
搜索 index.json（标签 + 关键词）
    ↓
搜索 preferences.md（全文）
    ↓
搜索 facts.md（全文）
    ↓
合并结果，按相关度排序
    ↓
返回 Top-5 匹配结果
```

## 记忆注入格式

检索到的记忆以以下格式注入到上下文：

```
[记忆上下文]
━━━━━━━━━━━━━━━━━━━━━━━━━
📋 偏好记录：
  · TypeScript，2 空格缩进
  · 仙侠小说风格

📌 关键事实：
  · 已安装 tianming-skill
  · 当前小说进度：第 3 章
━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 注入时机

| 时机 | 注入内容 |
|------|---------|
| 新对话开始 | 全部偏好 + 最新事实 |
| 用户问偏好 | 全部偏好 |
| 用户问进度 | 全部事实 |
| 用户说写代码 | 代码相关偏好 |
| 用户说写小说 | 小说相关偏好 |
